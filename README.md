# 📊 Arduino-Based Experimental Physics Data Acquisition System

## 📌 Περιγραφή έργου

Η παρούσα εργασία αφορά την ανάπτυξη και υλοποίηση μιας πειραματικής διάταξης για τη μελέτη φυσικών μεγεθών (απόσταση, ταχύτητα, επιτάχυνση) με χρήση μικροελεγκτή Arduino και αισθητήρα υπερήχων.

Το σύστημα συλλέγει δεδομένα από τον αισθητήρα, τα επεξεργάζεται σε πραγματικό χρόνο και τα αποστέλλει σε υπολογιστή για αποθήκευση και ανάλυση.

---

## 🧠 Αρχιτεκτονική συστήματος

Το σύστημα αποτελείται από τρία βασικά μέρη:

1. **Arduino Sketch**
   - Συλλογή δεδομένων από υπερηχητικό αισθητήρα
   - Υπολογισμός:
     - Απόστασης
     - Ταχύτητας
     - Επιτάχυνσης
   - Αποστολή δεδομένων μέσω σειριακής επικοινωνίας

2. **Εφαρμογή Visual Basic**
   - Λήψη δεδομένων από σειριακή θύρα
   - Εμφάνιση σε πραγματικό χρόνο
   - Αποθήκευση δεδομένων σε αρχείο CSV

3. **Vernier Graphical Analysis**
   - Εισαγωγή αρχείου CSV
   - Οπτικοποίηση δεδομένων μέσω γραφημάτων:
     - x–t (θέση–χρόνος)
     - v–t (ταχύτητα–χρόνος)
     - a–t (επιτάχυνση–χρόνος)

## 🚀 Οδηγίες χρήσης

1. Φορτώστε τον κώδικα στο Arduino
2. Εκτελέστε την εφαρμογή Visual Basic
3. Επιλέξτε COM port και φάκελο αποθήκευσης
4. Πατήστε "Start" για έναρξη καταγραφής
5. Εισάγετε το CSV στο Vernier Graphical Analysis για ανάλυση
